# Telegram to Google Sheets Automation

## Overview
An automation workflow that captures data from Telegram and stores it automatically in Google Sheets.

## Technologies
- n8n
- Telegram Bot API
- Google Sheets API

## Features
- Receive Telegram data
- Automatically store information in Google Sheets
- Automated notifications and workflows
